# Dessert Shop

A modern web application for browsing and ordering delicious desserts. Built with TypeScript and Vite.

## Features

- Browse a variety of desserts with images and prices
- Add items to your shopping cart
- View and manage cart contents
- Responsive design for mobile and desktop

## Technologies Used

- **TypeScript**: For type-safe JavaScript development
- **Vite**: Fast build tool and development server
- **HTML/CSS**: For structure and styling

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd dessertShop
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Usage

- Browse the dessert catalog
- Click "Add to Cart" on any dessert to add it to your cart
- View your cart by clicking the cart icon
- Adjust quantities or remove items as needed
- Proceed to checkout when ready

## Project Structure

- `src/components/`: Reusable UI components
- `src/data/`: Dessert data and configuration
- `src/logic/`: Business logic for cart and orders
- `src/types/`: TypeScript type definitions
- `src/utils/`: Utility functions for DOM manipulation and formatting
